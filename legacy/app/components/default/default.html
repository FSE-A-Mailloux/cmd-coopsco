Outil de gestion des commandes coopérative scolaire Auguste Mailloux

Ce site est un outil permettant de faciliter la gestion des commandes de fourniture scolaire pour les familles.

Les familles doivent se connecter au site via le formulaire en haut à droite.

Si vous n'avez pas encore de compte, il suffit de cliquer sur le lien "Créer compte" également en haut à droite, et de suivre les étapes.
